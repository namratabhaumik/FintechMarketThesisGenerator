# Exporting and sharing

Every thesis carries an export bar next to its job id. Four actions, no configuration:

| Action | What you get |
| --- | --- |
| **Copy as text** | The full thesis as plain text on your clipboard - scores, sources, themes, risks, signals. Pastes cleanly into notes, chat, or email. |
| **Download Markdown** | The same content as a `.md` file (`finthesis-<job_id>.md`), ready for a wiki or repo. |
| **Export PDF** | Opens the browser print dialog with a print-optimized layout: app controls (refine panel, buttons, library) are stripped, content sections remain. Save as PDF from there. |
| **Copy link** | The shareable `?job_id=...` URL for this thesis. Restores the full live state for anyone signed in to the owning account. |

Buttons confirm inline ("Copied", "Downloaded", "Link copied") without leaving the page.

!!! tip "Link vs. file"
    The link restores the *live* job - including refinement state and any later changes. The text/Markdown/PDF exports are snapshots of the thesis at the moment you export. For a record of what you approved, export after approving.
